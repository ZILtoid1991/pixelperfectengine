# 7 Bit Universal MIDI Sysex commands

|Code    |Function                                                                                               |
|--------|-------------------------------------------------------------------------------------------------------|
|0x00    |Null command                                                                                           |
|0x01 nn |Suspend channel nn while keeping its statuses.                                                         |
|0x02 nn |Resume channel nn without resetting its statuses.                                                      |
